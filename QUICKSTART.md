## Get Started in 3 Steps

### 1. Start the Application

```bash
cd dogcrap-tracker
npm run dev
```

Open your browser to: **http://localhost:5173**

### 2. Choose Your Profile

Click on **Chris** or **Denis** in the top-right corner.

### 3. Start Your First Workout

1. Click **"+ New Workout"**
2. Select Plan A or Plan B
3. Click **"Start Workout"**
4. Add exercises and track your sets!
